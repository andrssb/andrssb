## Sobre mim

Olá, meu nome é **Andres Barbosa**.

Estou em transição de carreira para a tecnologia, com background em vendas — o que me deu visão de negócio, comunicação e foco em resolver problemas reais. Agora aplico isso construindo software.

**Foco atual:** desenvolvimento full-stack. Construo APIs com Python/FastAPI, modelo bancos de dados e crio interfaces web — e venho expandindo para **Java (Spring Boot)** no backend e **Flutter** no cliente. Gosto de escrever código testado e bem organizado.

## Projetos em destaque

**[★ Castelo Amaldiçoado](https://github.com/andrssb/castelo-amaldicoado)** — Jogo de plataforma/ação medieval (inspirado nos clássicos estilo *Ghouls'n Ghosts*) construído **full-stack**: cliente do jogo em **Flutter + Flame** e servidor em **Java + Spring Boot**. O servidor gera um desafio diário procedural, mantém um ranking global e **valida cada partida no back-end (anti-cheat)**. Tem dois estágios com enigma, portal para passagem secreta e chefões. `Flutter`, `Flame`, `Java`, `Spring Boot`, `PostgreSQL`.

**[Minecraft Server Status](https://github.com/andrssb/minecraft-server-status)** — Painel web que consulta o status de qualquer servidor de Minecraft em tempo real (jogadores, versão, MOTD, ping), usando o protocolo Server List Ping do próprio jogo. `Python`, `FastAPI`, `mcstatus`.

**[Raio-X do Texto](https://github.com/andrssb/raio-x-do-texto)** — Analisador de estilo de escrita: detecta repetições, advérbios em excesso, frases longas, clichês e voz passiva. `Python`, `FastAPI`, `regex`.

**[Controle de Gastos](https://github.com/andrssb/controle-de-gastos)** — App de finanças pessoais com banco de dados e gráfico de gastos por categoria. `Python`, `FastAPI`, `SQLAlchemy`, `SQLite`.
